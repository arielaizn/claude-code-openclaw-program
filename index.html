Claude Code — תוכנית הלימודים // FROM ZERO TO HERO
// PHASE 01
🚀 התקנה וסביבת עבודה
שיעור 1-2
1
הכנת הסביבה
התקנת Node.js, npm, ואימות שהכל עובד. מה ההבדל בין כל מערכת הפעלה.
2
התקנת Claude Code
הרצת npm install -g @anthropic-ai/claude-code. חיבור לחשבון Claude — Premium vs API, מה עדיף לך.
3
הפרויקט הראשון
מריצים claude בטרמינל, יוצרים תיקייה לעסק, ומבינים איך לדבר עם הסוכן בצורה נכונה (CLAUDE.md).
4
חיבור לאנטיגרביטי (Antigravity)
הורדת Antigravity, התקנת תוסף Claude Code, ואיך משתמשים בהם ביחד — Gemini לתכנון, Claude לביצוע.
# התקנת Claude Code npm install -g @anthropic-ai/claude-code # חיבור לחשבון claude # → ייפתח browser לאימות # הרצה ראשונה mkdir my-business && cd my-business claude "תעזור לי לבנות מערכת ניהול לקוחות"
// PHASE 02
🧠 Skills — הכלי הכי עוצמתי
שיעור 3-5

Skills הם "מוחות מיוחדים" שאתה נותן ל-Claude Code — קבצי הוראות שמלמדים אותו לעשות משהו ספציפי בצורה מושלמת. ניצור skills לעסק שלך.

📁 my-business/

📄 invoice-generator.md ← skill ליצירת חשבוניות
📄 social-media.md ← skill לתוכן שיווקי
📄 CLAUDE.md ← חוקי ה-AI לעסק שלך
✍️
Skills שנבנה ביחד
skill לכתיבת הצעות מחיר, skill למענה ללקוחות, skill ליצירת תוכן ברשתות חברתיות — הכל בהתאם לעסק שלך.
📦
ייבוא Skills קיימים
יש אלפי skills מוכנים מהקהילה — נלמד איך למצוא, להתאים ולהשתמש ב-skills שכבר קיימים.

// PHASE 03
🤖 Sub-Agents — צוות AI שלם
שיעור 6-8

Sub-agent הוא Claude Code נפרד שמבצע משימות ספציפיות בשבילך. אתה יכול להפעיל 10 agents בו-זמנית — כל אחד עובד על משהו אחר.

📧
Agent אימיילים
קורא את תיבת הדואר, מסווג, מנסח תשובות ושולח — בעצמו.
📊
Agent דוחות
מושך נתונים, יוצר דוח שבועי/חודשי ושולח לך ב-WhatsApp.
📱
Agent תוכן
כותב פוסטים לאינסטגרם ולינקדאין, מתזמן פרסום — אוטומטי.
# דוגמה להפעלת 3 agents במקביל claude --agent "analyze sales data for this week" claude --agent "write 5 instagram posts based on our products" claude --agent "draft replies to all unanswered emails" # כולם רצים בו-זמנית!

OpenClaw — ה-AI שלך ב-WhatsApp // VPS SETUP GUIDE
🦞
מה זה OpenClaw? סוכן AI בקוד פתוח שרץ על שרת פרטי שלך — ומגיב לך ב-WhatsApp, Telegram, Discord. הוא מחובר ל-Claude, זוכר אותך, מריץ skills ו-agents, ועושה דברים בשבילך גם כשאתה ישן.
🔒
פרטיות מלאה
הכל על השרת שלך. הנתונים לא עוברים לאף שרת חיצוני. שולט על כל דבר.
📲
מ-WhatsApp בלבד
שולחים הודעה ב-WhatsApp → OpenClaw מריץ → מגיב עם תוצאות. פשוט כמו לדבר עם עוזר.
STEP 1
🖥️ רכישת שרת VPS
A
איפה לקנות?
DigitalOcean, Hetzner, Vultr, Linode — כולן עובדות. ל-OpenClaw צריך מינימום 2GB RAM. Hetzner הכי זולה (€4/חודש). DigitalOcean הכי ידידותית למתחילים ($12/חודש).
B
מה לבחור?
Ubuntu 24.04 LTS (מומלץ). גיאוגרפיה: EU/US — לא משנה לתפקוד, קצת משנה לחוקי פרטיות.
C
גישה ל-VPS
SSH key setup — נלמד איך להתחבר לשרת בבטחה. חשוב לדעת את זה!
# התחברות ל-VPS ssh root@YOUR_SERVER_IP # עדכון המערכת apt update && apt upgrade -y
STEP 2
🦞 התקנת OpenClaw
1
התקנת Node.js
OpenClaw דורש Node.js 20+. מתקינים עם nvm לניהול נוח של גרסאות.
2
התקנת OpenClaw
פקודה אחת מתקינה הכל ומריצה wizard שמנחה אותך שלב אחרי שלב.
3
חיבור ל-WhatsApp
Wizard מציג QR code — סורקים עם הטלפון בדיוק כמו WhatsApp Web. מוכן!
4
חיבור Claude API
מכניסים Anthropic API key (מחשבון pro/max) — OpenClaw מחובר ל-Claude ורץ 24/7.
# התקנת Node.js curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash nvm install 20 # התקנת OpenClaw npm install -g openclaw@latest openclaw onboard # wizard אוטומטי # OpenClaw מחובר! שלח הודעה ב-WhatsApp 🦞
⚠️
טיפ אבטחה חשוב הגדר allowFrom עם מספר הטלפון שלך בלבד — כך רק אתה יכול לשלוח פקודות ל-OpenClaw. אל תשאיר open-to-the-world!
STEP 3
📦 ייבוא Skills מ-Claude Code

ה-skills שיצרת ב-Claude Code עובדים גם ב-OpenClaw! אותו מבנה SKILL.md — מעתיקים לשרת ו-OpenClaw מפעיל אותם מ-WhatsApp.

A
ארכיטקטורת Skills ב-OpenClaw
Skills יכולים להיות bundled (מובנים), global (לכל ה-agents) או workspace (לסביבה ספציפית).
B
העלאה לשרת
SCP או Git — מעלים את תיקיית ה-skills שלנו ל-VPS. OpenClaw מזהה אוטומטית.
C
שימוש מ-WhatsApp
שולחים הודעה: "תשתמש ב-skill כתיבת הצעת מחיר עבור לקוח X" — OpenClaw מריץ את ה-skill ומחזיר תוצאה.
# העלאת skills מ-Mac/PC לשרת scp -r ~/.claude/skills root@YOUR_SERVER_IP:~/.openclaw/skills/ # לחלופין, דרך Git git clone https://github.com/YOUR_REPO/my-skills ~/.openclaw/skills/
STEP 4
🤖 הקמת Sub-Agents ב-OpenClaw

Sub-agents ב-OpenClaw הם agents נפרדים שפועלים בשמך — כל אחד עם workspace, skills ו-persona אחרת.

🧑‍💼
Agent ראשי
מקבל הכל ממך — מנתב לאיזה agent לשלוח ומסכם תוצאות.
📊
Agent נתונים
מנתח דוחות, מחשב מכירות, מכין גרפים — שולח לך סיכום ב-WhatsApp.
✉️
Agent תקשורת
מנהל אימיילים, שולח הודעות, מזכיר לך אירועים חשובים.
# config.yml — הגדרת agents agents: - name: "business-manager" model: "claude-opus-4-6" skills: ["reports", "crm", "invoices"] whatsapp: "+972XXXXXXXXX" - name: "content-agent" model: "claude-sonnet-4-6" skills: ["social-media", "copywriting"] heartbeat: "every: 24h"
STEP 5
🔗 חיבור למערכות שבנינו ב-Claude Code

OpenClaw יכול לתקשר עם מערכות שבנינו ב-Claude Code — API calls, webhooks, גישה לבסיסי נתונים.

🔌
Webhook Integration
שולחים הודעה ב-WhatsApp → OpenClaw קורא API של המערכת שבנינו → מחזיר נתונים חיים.
🗄️
גישה ל-Database
OpenClaw מחובר ישירות ל-Supabase/Postgres — יכול לקרוא ולכתוב נתונים ב-WhatsApp.
📡
MCP Servers
חיבור ל-Google Calendar, Gmail, Notion, Monday ועוד 50+ שירותים — דרך MCP.
⚡
Heartbeats
OpenClaw שולח לך עדכונים אוטומטיים כל X שעות — "מכירות היום: ₪12,400 ✅"
